# Shosh Digital — Agency Website

Official website for **Shosh Digital**, a full-service digital agency based in Tashkent, Uzbekistan.

🌐 **Live site:** [abdullohkariev.github.io/ShoshDigital](https://abdullohkariev.github.io/ShoshDigital/)
📦 **Repository:** [github.com/abdullohkariev/ShoshDigital](https://github.com/abdullohkariev/ShoshDigital)

---

## About

Shosh Digital builds brands that grow fast — websites, mobile apps, digital marketing, branding, and business automation.

---

## Features

- Multilingual — Uzbek, Russian, English
- Fully responsive (mobile + desktop)
- Contact form that opens Telegram with pre-filled message
- Smooth scroll reveal animations
- No frameworks — pure HTML, CSS, JavaScript

## Stack

| Layer | Tech |
|-------|------|
| Frontend | HTML, CSS, Vanilla JS |
| Hosting | GitHub Pages |
| Contact | Telegram (`@shoshdigital`) |
